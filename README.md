# SmartMailbox

This project consists of a client and server application for a smart mailbox system.

## Components

*   **[Client](./client/README.md)**: A Python script running on a Raspberry Pi that captures images and sends them to the server.
*   **[Server](./server/README.md)**: A Python server that processes images to identify mail recipients.

See the README in each directory for more detailed information on setup and usage.
